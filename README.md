🚗 Uplift – Ride Sharing System
📌 Description

Uplift is a Java-based desktop Ride Sharing System designed to simulate a real-world transportation platform. The system enables efficient coordination between riders and passengers through both instant and scheduled ride services.
It is built using:
JavaFX for the frontend
Spring Boot for the backend
MySQL for database management
The project follows Object-Oriented Programming (OOP) principles to ensure modularity, scalability, and maintainability.

🎯 Motivation

With the rapid growth of urban transportation needs, traditional systems often lack flexibility, transparency, and efficiency.
This project was developed to:
Model a real-world ride-sharing system
Improve practical skills in Java, JavaFX, Spring Boot, and databases
Demonstrate how software engineering principles solve real-life problems

🎯 Objectives
Develop a functional ride-sharing system with instant and scheduled rides
Apply core OOP concepts:
Encapsulation
Inheritance
Polymorphism
Abstraction
Design a role-based system (Admin, Rider, Passenger)
Ensure secure frontend-backend communication using APIs
Build a user-friendly JavaFX interface
Create UML diagrams for system design

✨ Features
👤 User Roles
Admin

Post scheduled rides

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/ar-hridoy/Uplift.git
cd Uplift
2️⃣ Backend Setup (Spring Boot)
cd uplift-backend
Configure MySQL database in application.properties
Run the Spring Boot application
3️⃣ Frontend Setup (JavaFX)
cd uplift-desktop
Open project in IDE (IntelliJ / Eclipse)
Run JavaFX application
